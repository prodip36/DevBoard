Question-1: What are the different ways to select an element in the DOM?

These methods allow you to select elements based on their ID, class, tag name, or other attributes.

  1. By ID: Use getElementById()
  2. By class: Use getElementsByClassName() or querySelector()
  3. By tag name: Use getElementsByTagName() or querySelector()
  4. By attributes or complex selectors: Use querySelector()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The main difference is that innerHTML returns the HTML markup as a string, innerText returns visible text content while considering CSS styling and layout, and textContent returns the raw text content of an element and all its descendants, ignoring styling and hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a JavaScript technique where you attach a single event listener to a parent element instead of multiple listeners to its child elements. It works by leveraging the concept of event bubbling, where an event triggered on a child element "bubbles up" the DOM tree to its ancestors

Question-4: What is event bubbling in the DOM?

Event bubbling is a method of DOM event propagation where an event triggered on a specific element first runs its event handler on that element, then on its parent element, and so on, up the chain of ancestors until it reaches the root of the document. This means that when a click occurs on a child element, the event is also "bubbled up" and can be handled by parent elements.
